# React simple blog 



Inorder to run the app in local machine

 1. download the code
 2. run the following comments

    2.1. To start the  server

    ```bash
        cd server
        change .env.expmle to .env
        npm install
        npm start
    

    ```

    2.2. to start client

    ``` bash
        cd client
        npm install
        npm start
    ```
